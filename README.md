Different tools have different timeout multipliers based on their complexity:

Tools are automatically categorized and can be filtered:
- **Passive**: subfinder, assetfinder, subdog
- **Active**: amass, bbot
- **Bruteforce**: dnscan
- **AI**: subwiz
- **Directory**: ffuf
